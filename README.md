# Fatherade Expense Tracker

> Track your income/expenses, and make an healthy budget

## Usage

```
change config.env file in config folder
```

```
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 yarn watch
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
